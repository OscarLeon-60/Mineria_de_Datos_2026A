# 📊 Minería de Datos 2026A

Repositorio académico correspondiente a la asignatura **Minería de Datos** del semestre **2026A**.

Este espacio contiene los trabajos, talleres y proyectos desarrollados durante el curso, aplicando técnicas de análisis de datos, modelado y aprendizaje automático.

---

## 📚 Descripción de la Asignatura

La materia de **Minería de Datos** se enfoca en:

- Limpieza y preparación de datos  
- Análisis exploratorio  
- Técnicas de clasificación y clustering  
- Modelos predictivos  
- Evaluación de modelos  
- Aplicación práctica de algoritmos de aprendizaje  

El contenido y la metodología están basados en el **Syllabus oficial** y el **Acuerdo Pedagógico** establecidos por la institución.

---

## 📝 Sistema de Evaluación

La asignatura se evalúa en **tres cortes académicos**:

### 🔹 Corte 1
- Fundamentos teóricos  
- Talleres prácticos  
- Actividades evaluativas  

### 🔹 Corte 2
- Aplicación de técnicas de minería de datos  
- Desarrollo de ejercicios prácticos  
- Avances de proyecto  

### 🔹 Corte 3
- Proyecto final integrador  
- Sustentación  
- Aplicación completa de los conocimientos adquiridos  

La nota definitiva se obtiene según la ponderación establecida en el syllabus oficial.
